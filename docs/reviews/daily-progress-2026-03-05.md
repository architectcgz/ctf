## 🔍 关键问题与解决方案

### 1. 配置硬编码问题（多次出现）

**问题**：多个任务中存在硬编码配置值，违反项目规范

**解决方案**：
- 统一在 `config.go` 中定义配置结构体
- 在 `config.yaml` 中提供配置示例
- Service 层从配置注入读取，移除所有硬编码

**涉及任务**：B10、B12、B13

---

### 2. 资源泄漏问题

**问题**：
- B9：Engine.Close() 未调用
- B12：清理逻辑只更新数据库，未释放 Docker 资源

**解决方案**：
- 在 main.go 中注册 graceful shutdown
- 清理逻辑中调用 container engine 删除容器和网络

---

### 3. 并发安全问题

**问题**：
- B11：端口管理器持锁期间执行网络 IO
- B12：延时操作存在 TOCTOU 竞争

**解决方案**：
- 端口管理器：锁分离策略，检测移到锁外
- 延时操作：使用原子更新（事务 + 乐观锁）

---

### 4. Docker 客户端注入问题

**问题**：B13 路由层传 `nil`，导致镜像验证功能失效

**解决方案**：
- Service 构造函数添加 `cfg *config.Config` 参数
- 路由注册时正确注入配置
- 预留 Docker 客户端注入接口
