#### [M1] 端口管理器未初始化随机数种子 ⚠️ 未修复

- **当前代码**：`internal/module/container/port_manager.go:19-25`

- **影响**：多次重启后端口分配可预测，可能与旧容器端口冲突

  ```go
      return &PortManager{
          rangeStart: start,
          rangeEnd:   end,

      rand.Seed(time.Now().UnixNano())
  }
  ```

#### [M2] 缺少端口范围合法性校验 ⚠️ 未修复
- **状态**：仍存在问题
- **问题**：`NewPortManager` 未校验 `start < end` 和端口在 1-65535 范围内
- **影响**：配置错误时可能导致 panic 或死循环
- **修正建议**：
  ```go
  func NewPortManager(start, end int) (*PortManager, error) {
      if start < 1 || start > 65535 || end < 1 || end > 65535 {
          return nil, fmt.Errorf("端口范围必须在 1-65535 之间")
      }
      if start >= end {
          return nil, fmt.Errorf("起始端口必须小于结束端口")
      }
      return &PortManager{
          rangeStart: start,
          rangeEnd:   end,
          usedPorts:  make(map[int]bool),
      }, nil
  }
  ```
