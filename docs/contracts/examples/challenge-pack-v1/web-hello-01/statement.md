# Web-01 Hello Container

## 题目描述

这是一个用于演示 `challenge-pack-v1` 题目包格式的样例题，不强调漏洞点。

平台启动实例后，会把一个 **动态 Flag** 注入到容器运行环境变量 `CTF_FLAG`（示例约定，具体以平台实现为准）。

你的任务是访问靶机服务，读取页面上显示的 `CTF_FLAG`，并在平台提交。

## 访问方式

- 访问地址：由平台实例创建接口返回（Host 动态端口映射）
- 默认路由：`/`

## 注意

- 真实题目不建议把 Flag 直接显示在首页，这里只是为了演示“容器环境配置”如何通过题目包描述。
