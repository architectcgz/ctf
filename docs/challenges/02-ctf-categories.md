# CTF 题目分类与解题思路

> 版本：v1.0 | 日期：2026-03-03 | 状态：整理中
> 来源：多个 CTF 平台与社区资料整理

---

## 0. 说明

CTF 竞赛通常将题目分为多个类别，每个类别考察不同的安全技能。本文档整理常见题目分类、所需技能、常用工具和解题思路。

---

## 1. Web 安全（Web Exploitation）

### 1.1 题目特点

- 分析和攻击 Web 应用、网站、Web 服务
- 识别并利用漏洞获取未授权访问或敏感信息
- 需要批判性思维和创造性思维

### 1.2 常见漏洞类型

- **SQL 注入**：通过输入恶意 SQL 语句操纵数据库
- **XSS（跨站脚本）**：注入恶意脚本到网页中
- **CSRF（跨站请求伪造）**：诱导用户执行非预期操作
- **文件包含漏洞**：LFI/RFI，读取或执行任意文件
- **命令注入**：通过输入执行系统命令
- **认证绕过**：绕过登录验证机制
- **目录遍历**：访问未授权的文件和目录
- **SSRF（服务端请求伪造）**：利用服务器发起内网请求

### 1.3 常用工具

- **Burp Suite**：Web 应用安全测试工具
- **OWASP ZAP**：自动化漏洞扫描
- **sqlmap**：自动化 SQL 注入工具
- **Nikto**：Web 服务器扫描器
- **curl/wget**：命令行 HTTP 客户端
- **浏览器开发者工具**：查看请求、响应、Cookie、Storage

### 1.4 解题思路

1. 信息收集：查看源代码、robots.txt、sitemap.xml
2. 功能测试：测试所有输入点（表单、URL 参数、Cookie、Header）
3. 漏洞识别：尝试常见 Payload，观察响应变化
4. 漏洞利用：构造精确 Payload 获取 flag
5. 绕过防护：WAF 绕过、编码变换、大小写混淆

## 2. 密码学（Cryptography）

### 2.1 题目特点

- 破解加密算法、分析密码系统
- 需要数学基础和密码学知识
- 从古典密码到现代加密算法

### 2.2 常见题目类型

- **古典密码**：凯撒密码、维吉尼亚密码、栅栏密码、摩斯密码
- **编码识别**：Base64、Base32、十六进制、URL 编码
- **对称加密**：AES、DES、RC4 的弱实现或密钥泄露
- **非对称加密**：RSA 小指数攻击、共模攻击、Wiener 攻击
- **哈希碰撞**：MD5、SHA1 碰撞，彩虹表攻击
- **随机数预测**：伪随机数生成器（PRNG）的弱实现
- **侧信道攻击**：时间攻击、能量分析

### 2.3 常用工具

- **CyberChef**：在线编码/解码/加密/解密工具
- **RsaCtfTool**：RSA 攻击工具集
- **hashcat/John the Ripper**：密码破解工具
- **Python 库**：pycryptodome、gmpy2、sympy
- **Sage Math**：数学计算软件
- **factordb.com**：大整数分解数据库

### 2.4 解题思路

1. 识别加密/编码类型
2. 分析密文特征（长度、字符集、熵）
3. 寻找已知明文或密钥泄露
4. 尝试已知攻击方法
5. 编写脚本自动化破解

## 3. 逆向工程（Reverse Engineering）

### 3.1 题目特点

- 分析二进制程序、固件、协议
- 理解程序逻辑，找到验证机制
- 需要汇编语言和调试技能

### 3.2 常见题目类型

- **静态分析**：反汇编代码，理解算法逻辑
- **动态调试**：运行时分析，修改内存/寄存器
- **反混淆**：去除代码混淆，还原原始逻辑
- **脱壳**：去除程序加壳保护
- **协议逆向**：分析网络协议或文件格式
- **Android/iOS 逆向**：移动应用分析
- **固件分析**：嵌入式设备固件提取与分析

### 3.3 常用工具

- **IDA Pro/Ghidra**：反汇编和反编译工具
- **x64dbg/gdb**：动态调试器
- **radare2**：开源逆向框架
- **Binary Ninja**：二进制分析平台
- **strings/file/objdump**：基础分析工具
- **ltrace/strace**：跟踪库调用和系统调用
- **Frida**：动态插桩框架

### 3.4 解题思路

1. 文件类型识别（file、strings）
2. 静态分析：找到关键函数（main、check、verify）
3. 动态调试：设置断点，观察执行流程
4. 算法还原：理解验证逻辑
5. 编写脚本或修改程序获取 flag

## 4. 二进制利用（Pwn/Binary Exploitation）

### 4.1 题目特点

- 利用二进制程序漏洞获取 shell 或读取 flag
- 需要深入理解内存布局、汇编、操作系统
- 高难度，需要大量练习

### 4.2 常见漏洞类型

- **栈溢出**：覆盖返回地址，劫持控制流
- **堆溢出**：破坏堆结构，利用 malloc/free 机制
- **格式化字符串**：利用 printf 等函数读写任意内存
- **整数溢出**：整数运算导致的逻辑错误
- **UAF（Use After Free）**：使用已释放的内存
- **ROP（Return-Oriented Programming）**：绕过 NX 保护
- **Shellcode 注入**：注入并执行恶意代码

### 4.3 常用工具

- **pwntools**：Python 漏洞利用框架
- **ROPgadget/ropper**：ROP 链构造工具
- **checksec**：检查二进制保护机制
- **gdb + pwndbg/gef**：调试器增强插件
- **one_gadget**：查找 libc 中的 one-gadget
- **LibcSearcher**：根据泄露地址查找 libc 版本

### 4.4 解题思路

1. 检查保护机制（checksec）
2. 静态分析找漏洞点
3. 动态调试确认漏洞
4. 构造 Payload（泄露地址、ROP 链、Shellcode）
5. 获取 shell 或读取 flag

## 5. 取证分析（Forensics）

### 5.1 题目特点

- 从文件、内存、网络流量中提取隐藏信息
- 需要细心和耐心
- 涉及多种文件格式和隐写技术

### 5.2 常见题目类型

- **文件分析**：损坏文件修复、文件头修复、隐藏数据提取
- **图片隐写**：LSB、EXIF、像素值分析
- **音频隐写**：频谱分析、波形分析
- **内存取证**：内存镜像分析，提取进程、密码、文件
- **磁盘取证**：文件恢复、删除文件还原
- **流量分析**：pcap 文件分析，协议还原
- **日志分析**：从日志中找线索

### 5.3 常用工具

- **binwalk**：固件分析、文件提取
- **foremost/scalpel**：文件恢复
- **steghide/stegsolve**：图片隐写分析
- **Audacity**：音频分析
- **Volatility**：内存取证框架
- **Wireshark/tshark**：流量分析
- **Autopsy/FTK**：磁盘取证工具
- **exiftool**：元数据查看

### 5.4 解题思路

1. 文件类型识别
2. 检查元数据和文件头
3. 使用专用工具分析（图片/音频/内存/流量）
4. 寻找异常数据或隐藏信息
5. 提取并解码 flag

## 6. 杂项（Miscellaneous）

### 6.1 题目特点

- 不属于其他分类的题目
- 创意性强，考察综合能力
- 可能涉及编程、脑洞、OSINT 等

### 6.2 常见题目类型

- **编程题**：算法实现、自动化脚本
- **OSINT（开源情报）**：搜索引擎、社交媒体、公开数据库
- **区块链**：智能合约漏洞、交易分析
- **AI/ML**：对抗样本、模型逆向
- **游戏破解**：修改游戏逻辑或内存
- **脑洞题**：需要创造性思维

### 6.3 常用工具

- **Python/Bash**：编写自动化脚本
- **Google/Shodan**：信息搜索
- **Remix/Mythril**：智能合约分析
- **Cheat Engine**：游戏内存修改
- **各类在线工具**：根据题目需求选择

### 6.4 解题思路

1. 理解题目要求
2. 识别题目类型
3. 选择合适工具或方法
4. 编写脚本自动化（如需要）
5. 验证结果

## 7. 技能树与学习路径

### 7.1 新手入门（0-3 个月）

**必备基础**
- Linux 命令行基础
- Python 编程基础
- 网络协议基础（HTTP、TCP/IP）
- 基础密码学概念

**推荐练习**
- picoCTF 的简单题目
- Web 安全：SQL 注入、XSS 基础
- 密码学：古典密码、Base64 编码
- 取证：基础文件分析

### 7.2 进阶提升（3-12 个月）

**深入学习**
- 汇编语言（x86/x64）
- C/C++ 编程
- 操作系统原理
- 常见漏洞原理（OWASP Top 10）

**推荐练习**
- 逆向工程：简单 crackme
- 二进制利用：栈溢出基础
- Web 安全：SSRF、文件包含、反序列化
- 密码学：RSA 基础攻击

### 7.3 高级进阶（1 年以上）

**专精方向**
- 选择 1-2 个方向深入（Web/Pwn/Reverse/Crypto）
- 研究最新漏洞和攻击技术
- 参与真实漏洞挖掘
- 阅读安全论文

**推荐练习**
- 参加高难度 CTF（DEF CON、Pwn2Own）
- 内核漏洞利用
- 浏览器/虚拟机逃逸
- 高级密码学攻击

## 8. 通用解题技巧

### 8.1 信息收集

- 仔细阅读题目描述，注意关键词和提示
- 查看文件名、文件大小、时间戳
- 使用 `file`、`strings`、`binwalk` 等工具初步分析
- 搜索题目相关的已知漏洞或技术

### 8.2 工具使用

- 熟练掌握常用工具的基本用法
- 学会阅读工具文档和源码
- 根据需求组合多个工具
- 必要时自己编写工具

### 8.3 脚本编写

- Python 是 CTF 最常用的脚本语言
- 学会使用 pwntools、requests、pycryptodome 等库
- 自动化重复性操作
- 保存常用脚本模板

### 8.4 团队协作

- 分工明确，发挥各自优势
- 及时分享进展和发现
- 记录解题过程和思路
- 赛后复盘，总结经验

## 10. 附录：常用命令速查

### 10.1 文件分析
```bash
file <filename>              # 识别文件类型
strings <filename>           # 提取可打印字符串
binwalk <filename>           # 分析固件/提取文件
exiftool <filename>          # 查看元数据
hexdump -C <filename>        # 十六进制查看
```

### 10.2 网络工具
```bash
curl -X POST -d "data" <url> # HTTP 请求
nc <host> <port>             # 网络连接
nmap <target>                # 端口扫描
tcpdump -r <pcap>            # 流量分析
```

### 10.3 编码转换
```bash
base64 -d <file>             # Base64 解码
xxd -r -p <hex>              # 十六进制转二进制
iconv -f <from> -t <to>      # 字符编码转换
```

### 10.4 密码破解
```bash
john <hashfile>              # 密码破解
hashcat -m <type> <hash>     # GPU 加速破解
hydra -l <user> -P <pass>    # 在线爆破
```
