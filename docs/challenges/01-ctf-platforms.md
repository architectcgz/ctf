# CTF 挑战平台列表（2026）

> 版本：v1.0 | 日期：2026-03-03 | 状态：整理中
> 来源：[StationX CTF Challenges for Beginners](https://www.stationx.net/ctf-challenges-for-beginners/)

---

## 0. 说明

本文档整理了 15 个主流 CTF 挑战平台，按难度级别、题目类型、费用等维度分类，方便选择合适的练习平台。

---

## 1. 初学者友好平台

### 1.1 picoCTF

- **官方网站**：picoCTF
- **难度级别**：初学者 → 中级
- **题目类型**：逆向工程、破解、黑客攻击、解密
- **赛制**：Jeopardy 风格（独立题目，按分值计分）
- **费用**：免费
- **年龄要求**：至少 13 岁
- **特点**：
  - 教育导向，适合入门
  - 题目循序渐进，有详细提示
  - 全年开放，可随时练习

### 1.2 niteCTF

- **官方网站**：niteCTF
- **难度级别**：初学者 → 中级
- **题目类型**：资产分析、密码学、取证、Pwn、逆向、Web
- **赛制**：Jeopardy 风格
- **费用**：免费
- **组织方**：Cryptonite 团队
- **特点**：
  - 面向学生群体
  - 题目覆盖全面
  - 在线参与，无地域限制

## 2. 全难度级别平台

### 2.1 GoogleCTF

- **官方网站**：GoogleCTF
- **难度级别**：初学者 → 高级
- **题目类型**：Pwn、逆向、密码学、Web、沙箱逃逸
- **赛制**：Jeopardy 风格
- **费用**：免费
- **组织方**：Google
- **特点**：
  - 真实场景模拟
  - 题目质量高，贴近实战
  - 全球参与，竞争激烈

### 2.2 DEF CON CTF

- **官方网站**：DEF CON
- **难度级别**：初学者 → 高级
- **题目类型**：漏洞利用、数据包分析、Web、密码学、取证
- **赛制**：多种赛制（Jeopardy、Attack-Defense 等）
- **费用**：$440 USD（现场参会）
- **地点**：美国拉斯维加斯
- **特点**：
  - 全球最知名黑客大会
  - 多个 CTF 赛事同时进行
  - 现场氛围浓厚，社区活跃

### 2.3 PlaidCTF

- **官方网站**：PlaidCTF
- **难度级别**：全级别
- **题目类型**：Web、密码学、Pwn、逆向
- **赛制**：Jeopardy 风格（地图格式）
- **费用**：免费
- **特点**：
  - 年度在线竞赛
  - 题目多样化
  - 独特的地图式界面

### 2.4 0CTF/TCTF

- **官方网站**：0CTF/TCTF
- **难度级别**：全级别
- **题目类型**：Pwnable、逆向、Web、密码学
- **赛制**：在线竞赛
- **费用**：免费
- **特点**：
  - 挑战性强，题目多样
  - 吸引全球参与者
  - 中国顶级 CTF 赛事

### 2.5 ASIS CTF

- **官方网站**：ASIS CTF
- **难度级别**：全级别
- **题目类型**：Web、密码学、漏洞利用、取证、逆向
- **赛制**：在线竞赛
- **费用**：免费
- **特点**：
  - 适合不同技能水平
  - 题目覆盖广泛
  - 伊朗主办的国际赛事

- **题目类型**：Web、密码学、逆向、Pwn、取证

### 3.4 37C3 Potluck CTF

- **官方网站**：37C3 Potluck CTF
- **难度级别**：全级别
- **题目类型**：硬件、逆向、Web、Pwn、密码学
- **赛制**：在线竞赛

### 3.5 CTF Internacional MetaRed

- **官方网站**：CTF Internacional MetaRed

- **题目类型**：密码学、取证、硬件、OSINT、Pwn、逆向、Web

## 5. 平台选择建议

### 5.1 按技能水平选择

| 技能水平 | 推荐平台 |
|---------|---------|
| 完全新手 | picoCTF、niteCTF |
| 初级进阶 | GoogleCTF（简单题）、ASIS CTF |
| 中级 | PlaidCTF、0CTF/TCTF、SECCON |
| 高级 | DEF CON、Pwn2Own、FAUST CTF |

### 5.2 按题目类型选择

| 题目类型 | 推荐平台 |
|---------|---------|
| Web 安全 | GoogleCTF、PlaidCTF、Wargames.MY |
| 密码学 | ASIS CTF、0CTF/TCTF |
| 逆向工程 | picoCTF、zer0pts CTF、SECCON |
| 二进制利用 | Pwn2Own、FAUST CTF、DEF CON |
| 取证分析 | Insomni'hack、CTF Internacional MetaRed |
| 硬件安全 | 37C3 Potluck CTF |

### 5.3 按参赛方式选择

- **全年开放练习**：picoCTF
- **定期在线赛事**：GoogleCTF、PlaidCTF、0CTF/TCTF
- **现场参与**：DEF CON、Insomni'hack
- **团队攻防**：FAUST CTF
