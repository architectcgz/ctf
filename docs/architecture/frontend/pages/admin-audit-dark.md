# 页面设计：审计日志 (Dark Mode)

> 继承：../design-system/MASTER.md | 技术栈：Element Plus + Tailwind CSS

---

## 布局

```
┌──────────────────────────────────────────────┐
│  审计日志                                     │
│  [搜索] [用户▼] [操作类型▼] [时间范围]        │
├──────────────────────────────────────────────┤
│  时间 | 用户 | 操作 | IP | 详情               │
│  10:30 | admin | 创建挑战 | 192.168.1.1 | [▼]│
└──────────────────────────────────────────────┘
```

## 技术栈

**Element Plus：**
- `<ElTable>` - 日志表格
- `<ElInput>` - 搜索框
- `<ElSelect>` - 筛选器
- `<ElDatePicker>` - 时间范围
