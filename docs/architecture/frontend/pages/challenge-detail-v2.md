# 页面设计：靶场详情页 v2 整合视图 (Challenge Detail v2)

> 继承：../design-system/MASTER.md | 角色：学员 | 整合：challenges.md + hint-system.md + solve-timeline.md + writeup.md
> 技术栈：Element Plus (主) + Tailwind CSS (辅)
> 说明：Phase 2 后靶场详情页的完整布局，统一三个新增区块的信息架构

---

## 技术栈

**Element Plus：** `<ElTabs>`, `<ElCard>`, `<ElTag>`, `<ElButton>`, `<ElInput>`, `<ElTimeline>`
**Tailwind CSS：** 布局、间距、响应式

---

## 1. 整体布局

Phase 2 后靶场详情页采用 Tab 结构，避免单页信息过载：

```
┌──────────────────────────────────────────────────────────┐
│  面包屑: 靶场训练 > SQL注入基础                            │
├──────────────────────────────────────────────────────────┤
│  题目头部 (始终可见，不随 Tab 切换)                        │
│  SQL 注入基础                                   100 pts  │
│  [Web] [简单]  128人解出  [sql-injection] [owasp-top10]  │
├──────────────────────────────────────────────────────────┤
│  [题目详情] [解题记录] [Writeup]                          │
├──────────────────────────────────────────────────────────┤
│                                                          │
│  (当前 Tab 内容)                                         │
│                                                          │
└──────────────────────────────────────────────────────────┘
```

### Tab 可见性规则

| Tab | 可见条件 |
|-----|---------|
| 题目详情 | 始终可见（默认激活） |
| 解题记录 | 用户对该靶场有操作记录时显示 |
| Writeup | 已解出+已公开，或未解出+已公开 |

---

## 2. Tab：题目详情

整合 challenges.md 详情页 + hint-system.md：

```
┌──────────────────────────────────────────────────────────┐
│  题目描述 (Markdown 渲染)                                 │
├──────────────┬───────────────────────────────────────────┤
│ 靶机实例控制  │ Flag 提交                                  │
│ (复用 challenges.md 现有设计)                              │
├──────────────┴───────────────────────────────────────────┤
│  提示 (已解锁 1/3)                                       │
│  (复用 hint-system.md 设计)                               │
└──────────────────────────────────────────────────────────┘
```

布局顺序：题目描述 → 实例控制 + Flag 提交（并排）→ 提示系统

---

## 3. Tab：解题记录

完整复用 solve-timeline.md 设计，无额外修改。

---

## 4. Tab：Writeup

复用 writeup.md 学员端设计，包含剧透警告逻辑。

竞赛进行中时此 Tab 隐藏。

---

## 5. Tab 样式

```
复用 MASTER Tab 规范:
  默认: text-secondary hover:text-primary px-4 py-2
  活跃: text-primary border-b-2 border-primary

Tab 栏: border-b border-border bg-surface sticky top-14 z-10
  (sticky 定位，滚动时固定在顶部导航下方)
```
