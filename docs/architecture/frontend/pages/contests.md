# 页面设计：竞赛中心 (Contests)

> 继承：../design-system/MASTER.md
> 技术栈：Element Plus (主) + Tailwind CSS (辅)

---

## 技术栈使用

**Element Plus：**
- `<ElCard>` - 竞赛卡片
- `<ElTag>` - 状态/模式标签
- `<ElButton>` - 操作按钮
- `<ElTabs>` - 题目/公告/排行榜切换

**Tailwind CSS：**
- 布局：`space-y-4`, `flex justify-between`
- 响应式：`md:grid-cols-2`
