## 2. 提示卡片样式

### 已解锁

```
bg-surface border border-border rounded-lg p-4

标题行:
  flex justify-between items-center mb-2
  左: [CheckCircle 16px text-success] "Hint 1" text-sm font-medium text-primary
  右: "扣分: -10 pts" text-xs text-muted font-mono

内容:
  text-sm text-secondary leading-relaxed
```

### 未解锁

```
bg-surface border border-border rounded-lg p-4

标题行:
  左: [Lock 16px text-muted] "Hint 2" text-sm font-medium text-muted
  右: "扣分: -20 pts" text-xs text-warning font-mono

内容:
  模糊遮罩文字 (bg-border rounded h-4 的占位条，模拟内容)

解锁按钮:
  mt-3
  次要按钮 sm: "解锁提示 (-20 pts)"
```

### 已解出靶场回看

所有提示自动展开，解锁按钮隐藏，扣分信息保留。

## 3. 解锁确认弹窗

```
┌─────────────────────────────────────┐
│  "确认解锁提示"              [关闭]  │
├─────────────────────────────────────┤
│  解锁 Hint 2 将扣除 20 分。         │
│                                     │
│  当前得分: 100 pts                   │
│  已扣分:   -10 pts (Hint 1)         │
│  本次扣分: -20 pts                   │
│  ─────────────────                  │
│  解锁后得分: 70 pts                  │
│                                     │
│  [取消]          [确认解锁]          │
└─────────────────────────────────────┘
```

- Dialog 宽度：`max-w-sm`
- 得分数字：`font-mono`
- 扣分行：`text-warning`
- 解锁后得分：`text-lg font-bold font-mono`

## 5. 管理员端 — 提示配置

在靶场创建/编辑 Drawer 的"提示"步骤中增强：

```
┌─────────────────────────────────────────────┐
│  提示配置                                    │
│                                              │
│  Hint 1                                      │
│  内容: [注意登录表单的参数传递方式...]         │
│  扣分比例: [10▼] %                           │
│  [删除]                                      │
│                                              │
│  Hint 2                                      │
│  内容: [尝试使用单引号测试SQL注入...]          │
│  扣分比例: [20▼] %                           │
│  [删除]                                      │
│                                              │
│  [+ 添加提示]                                │
│                                              │
│  预览: 100分靶场，3级提示扣分 10+20+30=60分   │
│  最低可得分: 40 pts                           │
└─────────────────────────────────────────────┘
```

- 扣分比例下拉：5% / 10% / 15% / 20% / 25% / 30%
- 提示可拖拽排序（级别顺序）
- 预览行：`text-xs text-muted`，实时计算
