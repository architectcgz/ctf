# 页面设计：错误页面 (Dark Mode)

> 继承：../design-system/MASTER.md | 技术栈：Element Plus + Tailwind CSS

---

## 1. 404 页面

```
┌──────────────────────────────────────────────┐
│                                               │
│              [404图标]                        │
│           页面未找到                          │
│      您访问的页面不存在或已被删除              │
│           [返回首页]                          │
│                                               │
└──────────────────────────────────────────────┘
```

## 2. 403 页面

```
┌──────────────────────────────────────────────┐
│              [403图标]                        │
│           访问被拒绝                          │
│      您没有权限访问此页面                      │
│           [返回首页]                          │
└──────────────────────────────────────────────┘
```

## 技术栈

**Element Plus：**
- `<ElResult>` - 结果页组件
- `<ElButton>` - 返回按钮

**Tailwind CSS：**
- 居中布局：`flex min-h-screen items-center justify-center`
