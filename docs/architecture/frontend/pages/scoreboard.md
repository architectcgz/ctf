# 页面设计：排行榜 (Scoreboard)

> 继承：../design-system/MASTER.md
> 技术栈：Element Plus (主) + Tailwind CSS (辅)

---

## 技术栈使用

**Element Plus：**
- `<ElTable>` - 排行榜表格
- `<ElTag>` - 排名标记
- `<ElSelect>` - 竞赛选择

**Tailwind CSS：**
- 表格样式覆盖
- 间距：`space-y-6`

---

## 布局结构

```
┌──────────────────────────────────────────────┐
│  "排行榜"  [竞赛排行榜] [积分排行榜]             │
├──────────────────────────────────────────────┤
│  竞赛排行列表                                  │
│  当前竞赛     进行中  Jeopardy   [查看完整排行榜] │
│  春季 AWD     已封榜  AWD        [查看完整排行榜] │
│  往期竞赛     已结束  Jeopardy   [查看完整排行榜] │
├──────────────────────────────────────────────┤
│  积分排行榜页签直接展示全站练习积分榜             │
└──────────────────────────────────────────────┘
```

竞赛排行列表不直接展开队伍排名明细。点击某个竞赛后进入 `/scoreboard/:contestId`：

```
┌──────────────────────────────────────────────┐
│  [返回排行列表]                               │
│  "当前竞赛"                       [刷新]       │
├──────────────────────────────────────────────┤
│  榜单队伍 | 最高分 | 总解题 | 榜单状态          │
├──────────────────────────────────────────────┤
│  排行详情                                     │
│  # | 队伍 | 得分 | 解题数 | 最近得分            │
└──────────────────────────────────────────────┘
```

## 组件细节

### 竞赛排行列表
- 只加载可展示竞赛列表，不预取每个竞赛的完整榜单。
- 每个竞赛条目展示状态、赛制、比赛时间和“查看完整排行榜”入口。
- 运行中或封榜赛事的 WebSocket 实时刷新只在排行详情页挂载。

### 排名表格
- Top 3 行使用高亮样式（见 MASTER 表格规范）
- 排名数字：`font-mono font-bold`
- 得分数字：`font-mono`
- 当前用户/队伍所在行：`bg-[#0891b2]/5`

### 得分趋势图（竞赛模式）
- ECharts line chart，暗色主题
- 仅在查看竞赛排名时展示，全站排名不显示
- 每条线使用不同颜色区分队伍

### 冻结榜
- 表格上方横幅：`bg-warning/5 border border-warning/20 rounded-lg p-3`
- 文字："排行榜已冻结，最终排名赛后揭晓"
- 冻结期间的新提交显示 "?" 占位

### 排名变动
- 排名上升：`text-success` 小箭头
- 排名下降：`text-danger` 小箭头
- 行位移动画 300ms（仅竞赛实时模式）
