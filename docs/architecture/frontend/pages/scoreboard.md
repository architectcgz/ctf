# 页面设计：排行榜 (Scoreboard)

> 继承：../design-system/MASTER.md
> 技术栈：Element Plus (主) + Tailwind CSS (辅)

---

## 技术栈使用

**Element Plus：**
- `<ElTable>` - 排行榜表格
- `<ElTag>` - 排名标记
- `<ElSelect>` - 竞赛选择

**Tailwind CSS：**
- 表格样式覆盖
- 间距：`space-y-6`

---

## 布局结构

```
┌──────────────────────────────────────────────┐
│  "排行榜"  [全站排名] [竞赛排名▼]              │
├──────────────────────────────────────────────┤
│  排名表格                                     │
│  # | 队伍/用户 | 得分 | 解题数 | 最后提交      │
│  ─────────────────────────────────────────── │
│  1 | Binary Wizards  | 2450 | 8 | 11:32     │
│  2 | Null Pointers   | 2100 | 7 | 11:45     │
│  3 | Stack Overflow  | 1850 | 7 | 12:01     │
│  4 | ...                                     │
│  [分页]                                       │
├──────────────────────────────────────────────┤
│  得分趋势图 (仅竞赛排名时显示)                  │
│  ECharts Line，Top 5 队伍得分时间线             │
└──────────────────────────────────────────────┘
```

## 组件细节

### 排名表格
- Top 3 行使用高亮样式（见 MASTER 表格规范）
- 排名数字：`font-mono font-bold`
- 得分数字：`font-mono`
- 当前用户/队伍所在行：`bg-[#0891b2]/5`

### 得分趋势图（竞赛模式）
- ECharts line chart，暗色主题
- 仅在查看竞赛排名时展示，全站排名不显示
- 每条线使用不同颜色区分队伍

### 冻结榜
- 表格上方横幅：`bg-warning/5 border border-warning/20 rounded-lg p-3`
- 文字："排行榜已冻结，最终排名赛后揭晓"
- 冻结期间的新提交显示 "?" 占位

### 排名变动
- 排名上升：`text-success` 小箭头
- 排名下降：`text-danger` 小箭头
- 行位移动画 300ms（仅竞赛实时模式）
