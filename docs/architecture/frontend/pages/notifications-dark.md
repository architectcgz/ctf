# 页面设计：通知中心 (Dark Mode)

> 继承：../design-system/MASTER.md | 技术栈：Element Plus + Tailwind CSS

---

## 布局

```
┌──────────────────────────────────────────────┐
│  通知中心                    [全部标记已读]    │
├──────────────────────────────────────────────┤
│  ● [系统] 平台维护通知                        │
│    2024-03-15 10:00                          │
│  ○ [竞赛] 春季CTF即将开始                     │
│    2024-03-14 15:30                          │
└──────────────────────────────────────────────┘
```

## 技术栈

**Element Plus：**
- `<ElTimeline>` - 时间线
- `<ElBadge>` - 未读标记
- `<ElButton>` - 标记已读按钮

**Tailwind CSS：**
- 间距：`space-y-4`
