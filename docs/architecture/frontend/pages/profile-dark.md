# 页面设计：用户资料 (Dark Mode)

> 继承：../design-system/MASTER.md | 技术栈：Element Plus + Tailwind CSS

---

## 布局结构

```
┌──────────────────────────────────────────────┐
│  [头像] 用户名                                 │
│         学号 | 班级 | 加入时间                │
│  [编辑资料]                                   │
├──────────────────────────────────────────────┤
│  [统计] [解题记录] [成就]                      │
│  ─────────────────────────────────────────── │
│  内容区域                                     │
└──────────────────────────────────────────────┘
```

## 技术栈

**Element Plus：**
- `<ElAvatar>` - 头像
- `<ElTabs>` - 标签页切换
- `<ElTable>` - 解题记录表格
- `<ElButton>` - 编辑按钮

**Tailwind CSS：**
- Flex 布局：`flex items-center gap-4`
- 间距：`space-y-6`
