# 页面设计：管理后台 (Admin)

> 继承：../design-system/MASTER.md | 角色：管理员
> 技术栈：Element Plus (主) + Tailwind CSS (辅)

---

## 技术栈使用

**Element Plus：**
- `<ElTable>` - 数据表格
- `<ElDialog>` - 创建/编辑弹窗
- `<ElForm>` - 表单
- `<ElSelect>` - 下拉选择
- `<ElButton>` - 操作按钮

**Tailwind CSS：**
- Grid 布局：`grid grid-cols-2 md:grid-cols-4 gap-4`
- 间距：`space-y-6`

## 3. 竞赛管理

同靶场管理表格布局，字段：
- 竞赛名称、模式、状态（草稿/报名中/进行中/已冻结/已结束）
- 参赛队伍数、时间窗口
- 操作：编辑、发布、结束、删除
