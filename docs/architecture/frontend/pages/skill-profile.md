## 组件细节

### 雷达图
- min-height 280px
- 填充：`rgba(8, 145, 178, 0.12)`，边框 `#0891b2`
- 各维度标签使用对应分类色标

### 维度进度条
- `[分类色标圆点] [名称] [进度条] [百分比]`
- 进度条颜色跟随分类色标
- 低于 40%：进度条右侧显示 `text-warning` 警告图标

### 薄弱项提醒
- `bg-warning/5 border border-warning/20 rounded-lg p-4`
- Lucide AlertTriangle 图标

### 空状态
- 居中："完成你的第一道题，开始生成能力画像"
- 引导按钮跳转靶场列表
