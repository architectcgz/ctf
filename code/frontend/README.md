# CTF Frontend

基于 `ctf/docs/architecture/frontend` 初始化的前端工程骨架。

## 技术栈

- Vue 3 + `<script setup lang="ts">`
- Vite 7
- Vue Router 4
- Pinia 3
- Axios
- Element Plus
- Tailwind CSS 4

## 已就绪能力

- 路由骨架与角色守卫
- Pinia 全局状态
- Axios 请求封装与 401 刷新队列
- 登录/注册页与主布局壳
- 学员、教师、管理员页面占位视图

## 常用命令

```bash
npm run dev
npm run build
npm run typecheck
```
