# 页面设计：审计日志 (Audit Logs)

> 继承：design-system/MASTER.md | 角色：管理员 | 位置：管理后台子页
> 技术栈：Element Plus (主) + Tailwind CSS (辅)

---

## 技术栈

**Element Plus：** `<ElTable>`, `<ElInput>`, `<ElSelect>`, `<ElDatePicker>`, `<ElButton>`
**Tailwind CSS：** 布局、间距

---

## 布局结构

```
┌──────────────────────────────────────────────┐
│  "审计日志"                        [导出CSV]  │
│  [搜索用户] [操作类型▼] [时间范围▼]            │
├──────────────────────────────────────────────┤
│  表格                                         │
│  时间 | 用户 | 操作 | 目标 | IP | 结果         │
│  ─────────────────────────────────────────── │
│  03-01 14:32  admin   创建靶场  Web-05       │
│                       10.0.1.1   成功         │
│  03-01 14:30  admin   修改用户  zhangsan     │
│                       10.0.1.1   成功         │
│  03-01 14:28  lisi    提交Flag  Pwn-02       │
│                       10.0.1.22  失败         │
│  ─────────────────────────────────────────── │
│  [分页]                                       │
└──────────────────────────────────────────────┘
```

### 组件细节

- 时间列：`font-mono text-xs text-muted`
- 操作类型筛选：登录/登出、靶场管理、竞赛管理、用户管理、Flag 提交、实例操作
- 结果列：成功 `text-success` / 失败 `text-danger`
- IP 地址：`font-mono text-xs`
- 时间范围：预设（今天/近7天/近30天）+ 自定义日期选择
- 表格支持按时间倒序排列，默认显示最新记录
