## 3. 解题热力图

类似 GitHub 贡献图，按日统计解题数量。

### 色阶

```
0 道:  bg-[#161b22]  (同 surface，几乎不可见)
1 道:  bg-[#0891b2]/20
2 道:  bg-[#0891b2]/40
3 道:  bg-[#0891b2]/60
4+ 道: bg-[#0891b2]/80
```

### 布局

```
每格: w-3 h-3 rounded-sm gap-[2px]
行标签: 一/三/五/日  text-[10px] text-muted
列标签: 月份  text-[10px] text-muted
tooltip (hover): "2026-01-15: 解出 3 道题"  bg-elevated text-xs p-2 rounded
```

### 月份切换

```
右上角: [ChevronLeft] "2025-10 ~ 2026-03" [ChevronRight]
默认显示近 6 个月，可前后翻页
```

## 4. 分类完成率

### 进度条

```
每行: flex items-center gap-3 py-2

分类名: w-20 text-sm text-secondary
进度条: flex-1 h-2 bg-border rounded-full overflow-hidden
  填充: bg-{分类色} rounded-full  (使用 MASTER 分类色标)
  Web: bg-cyan-500, Crypto: bg-violet-400, Pwn: bg-red-500 ...
统计: w-24 text-right text-xs text-muted font-mono  "18/25  72%"
```

### 点击交互

点击分类行 → 跳转到靶场列表页，自动筛选该分类。
