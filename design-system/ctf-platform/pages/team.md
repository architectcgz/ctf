# 页面设计：组队管理 (Team Management)

> 继承：design-system/MASTER.md | 位置：竞赛比赛页内 Tab + 独立入口
> 技术栈：Element Plus (主) + Tailwind CSS (辅)

---

## 技术栈

**Element Plus：** `<ElCard>`, `<ElInput>`, `<ElButton>`, `<ElDialog>`, `<ElAvatar>`
**Tailwind CSS：** 布局、间距

### 组件细节

- 邀请码：`font-mono text-primary bg-primary/10 px-2 py-0.5 rounded`
- 队长标识：`text-xs bg-warning/10 text-warning px-1.5 py-0.5 rounded`
- 成员头像：`w-8 h-8 rounded-full bg-elevated` 显示首字母
- 空位：`border border-dashed border-border rounded-lg p-3 text-muted text-center`
- 退出/解散按钮：危险按钮样式，需二次确认 Dialog
- 队伍人数上限由竞赛配置决定，显示为 "成员 (当前/上限)"
