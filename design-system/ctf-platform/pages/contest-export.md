## 3. 异步导出状态

大数据量（200+ 参赛者）走后台异步任务。

### 导出中状态

```
点击 [开始导出] 后按钮变为 loading 态

导出历史列表顶部插入新记录:
  ┌────────────────────────────────────────────────────┐
  │ 竞赛成绩总表+解题明细  xlsx  --     03-01 16:05   │
  │ [Loader2 旋转 16px] 导出中... 预计 30s             │
  └────────────────────────────────────────────────────┘

状态标签:
  导出中: bg-primary/10 text-primary "导出中..."
  已完成: bg-success/10 text-success "已完成"
  失败:   bg-danger/10 text-danger "导出失败" + [重试]
```

### 完成通知

导出完成后通过通知中心推送，Toast 提示"数据导出完成，点击下载"。
