# 页面设计：通知中心 (Notifications)

> 继承：design-system/MASTER.md | 角色：所有用户
> 技术栈：Element Plus (主) + Tailwind CSS (辅)

---

## 技术栈

**Element Plus：** `<ElDropdown>`, `<ElBadge>`, `<ElTabs>`, `<ElEmpty>`
**Tailwind CSS：** 布局、间距、响应式

### 组件细节

- 未读行：左侧 `w-2 h-2 rounded-full bg-primary` 圆点
- 已读行：`text-secondary`
- 通知类型图标：竞赛=Trophy, 系统=AlertCircle, 解题=Flag, 靶机=Server
- 时间戳：`text-xs text-muted`，使用相对时间（刚刚/N分钟前/N小时前/昨天）
- 分类 Tab：带未读计数徽章 `bg-primary/10 text-primary text-xs rounded-full px-1.5`
- WebSocket 实时推送：新通知到达时列表顶部 slide-down 插入

### 空状态
- 图标：Lucide InboxIcon 48px
- 文案："暂无通知"
