## 3. 从模板创建靶场

点击模板卡片 [使用] 按钮后，跳转到靶场创建 Drawer，自动填充：
- 拓扑配置（节点、连线、网络层）
- 关联镜像（如模板中的镜像存在）
- 资源限制配置

顶部提示条：

```
bg-primary/5 border border-primary/20 rounded-lg px-4 py-2 text-sm
[Info 16px text-primary] "基于模板「Web三层渗透环境」创建，可自由修改配置"
```

## 4. 导入/导出

### 导出
- 更多菜单 > 导出 → 下载 JSON 文件
- 文件名：`template-{模板名}-{日期}.json`

### 导入
- [+ 导入模板] 按钮 → 文件选择器（仅 .json）
- 导入前校验：格式合法性、镜像引用检查
- 镜像不存在时提示：

```
bg-warning/5 border border-warning/20 rounded-lg p-4
[AlertTriangle 16px text-warning]
"模板引用了 2 个本地不存在的镜像，导入后需手动关联"
缺失镜像: nginx:1.25, mysql:8.0
[取消]  [仍然导入]
```
