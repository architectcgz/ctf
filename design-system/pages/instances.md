## 2. 排队等待状态

```
当实例启动遇到资源排队时，卡片内显示:
┌─────────────────────────────────────────┐
│ SQL 注入基础              [Web] [简单]   │
│ 状态: ◐ 排队中                          │
│ 队列位置: 第 3 位                        │
│ 预计等待: ~2 分钟                        │
│ ████████░░░░░░░░ 进度条                 │
│ [取消排队]                               │
└─────────────────────────────────────────┘
```

### 组件细节

- 实例上限提示：`text-sm text-secondary`，达到上限时 `text-warning`
- 状态指示：
  - 运行中：`text-success` 实心圆点
  - 即将过期（<5min）：倒计时 `text-warning`
  - 排队中：`text-primary` 半圆动画
  - 已过期：`text-muted` 空心圆点
  - 启动失败：`text-danger` + 错误信息
- 地址：`font-mono text-sm`，复制按钮 Lucide Copy
- 倒计时：`font-mono`，每秒更新
- 排队进度条：`bg-primary/20` 底色 + `bg-primary` 填充，轮询更新位置
