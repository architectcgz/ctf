# Reverse 入门：小小校验器题解

## 解法

`checker.py` 的关键逻辑是：

```python
return "".join(chr(x ^ 0x7A) for x in data)
```

说明程序把 `data` 中每个数字与 `0x7A` 异或后拼成字符串。直接运行恢复逻辑即可：

```bash
python3 - <<'PY'
data = [
    0x1c, 0x16, 0x1b, 0x1d, 0x01, 0x08, 0x1f, 0x0c, 0x1f, 0x08, 0x09,
    0x1f, 0x25, 0x18, 0x1b, 0x18, 0x03, 0x25, 0x19, 0x12, 0x1f, 0x19,
    0x11, 0x1f, 0x08, 0x07,
]
print("".join(chr(x ^ 0x7A) for x in data))
PY
```

得到 flag：

```text
flag{reverse_baby_checker}
```
