团队每周会把 Agent 冲刺周报导出成 Markdown 发给同事，最近有人怀疑文档里被夹带了“看不见的水印”。

页面提供了一份可下载的 `briefing.md` 和一个兑换接口。请从这份 Markdown 里恢复隐藏内容，并拿到实例中的动态 Flag。

## 目标

1. 下载 `briefing.md`。
2. 从不可见字符中恢复兑换码。
3. 调用兑换接口拿到动态 Flag。

## 获取方式

- `/` 首页
- `/download/briefing.md`
- `/redeem?code=<value>`

## 补充说明

- 可见文本是正常业务周报。
- 隐藏数据不需要额外解压。
