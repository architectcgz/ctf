团队把本地 Agent 的 cache 兑换逻辑单独编成了一个 Python helper，准备用于内网排障。

现在页面只保留了一个编译后的 `pyc` 样本和一个兑换接口。请分析这份 helper，恢复出正确的兑换码，并拿到实例中的动态 Flag。

## 目标

1. 下载页面提供的 `pyc` 样本。
2. 从字节码中恢复兑换码。
3. 调用兑换接口拿到动态 Flag。
3. 在平台提交拿到的动态 Flag。

## 获取方式

- `/` 首页
- `/download/cache_gate.pyc` 下载样本
- `/redeem?code=<value>` 兑换接口

## 补充说明

- 不需要联网查题，样本本身已经足够恢复兑换码。
- 字节码来自 Python 3.12。
